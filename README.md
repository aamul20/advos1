# Advanced Operating System Management Systems

This repository contains the implementation of three system administration and management tools developed for the Advanced Operating Systems module. The project demonstrates the integration of **Bash** and **Python** to solve complex OS-level challenges.

## Repository Structure
* **`Task1/`**: Contains the Bash script for the Data Centre Admin Tool.
* **`Task2/`**: Contains the Python script for the High-Performance Computing (HPC) Job Scheduler.
* **`Task3/`**: Contains the standalone Python and Bash scripts for the Secure Examination Submission System.

## Task 1: University Data Centre Admin Tool (Bash)
A POSIX-compliant Bash script designed to monitor shared Linux server resources.
or traceability.

**How to run:**
bash
./Task1/task1.sh


## Task 2: High Performance Computing Job Scheduler (Python)
A Python-based scheduling engine simulating a university computational cluster.


**How to run:**

python Task2/scheduler.py


## Task 3: Secure Submission & Access Control (Python & Bash)
Two standalone scripts (one Python, one Bash) demonstrating equivalent security controls for an Examination Board submission vault.

**How to run (Python version):**

python Task3/task3.py

**How to run (Bash version):**
bash
./Task3/secure_admin.sh
